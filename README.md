web-приложение на реакт

сайт, где можно сформировать список покупок, поделиться этим листом, создать объединенный список

интерфейс на mui

хранение данных - mongodb

Связь с сервером через api

Можно создавать индивидуальные, ограниченно доступные или общедоступные списки

Списки для друзей сразу попадают в их аккаунт

Организован поиск общедоступных списков. Любой пользователь может найти его по ID

Любым списком можно поделиться с любым человеком сгенерировав URL

Можно выбрать режим фона. Салютики на готовой библиотеке, летающие точки на threeJS

В настройках можно включить неоновую надпись и анимации

Валидация e-mail имеется

имеется возможность загружать аватары

Объединенные списки формируются при первом рендеринге страницы. Условия формирования хранятся в локале

Пароли шифруются. Информация пересылается с использованием JWT-токена
